# Project
Website from DH design
Xây dựng công cụ tìm kiếm truyện trực tuyến hiệu quả có khả năng xử lí chính xác các từ khóa Tiếng Việt có dấu, công cụ này giúp người dùng tìm kiếm truyện theo tên, thể loại, tác giả và các yếu tố khác 1 cách chính xác, tiết kiệm thời gian hơn. Ngoài ra còn tích hợp thêm trí tuệ nhân tạo vào trang web nhằm tương tác với người đọc và thực hiện một số nhiệm vụ hỗ trợ người dùng
 ####
 Công nghệ sử dụng
- Front End: HTML, CSS(Tailwild), React.js
- Back End: Node.js, Express/Nest.JS
- Database: PostgreSQL 
- AI features: OpenAI API
- Realtime: Socket.io ( Sử dụng cho chức năng nâng cao nhân vật ảo)
